Placify - Placement Management System

Project Details

Placify is a web-based Placement Management System designed to streamline the recruitment process between students, educational institutions, and recruiters. Built using PHP and MySQL, the platform offers role-based access for students, recruiters, and admins, enabling seamless registration, job posting, and application workflows.

Students can register, build their profiles with academic and personal data, upload resumes, browse job postings, and track application status with real-time updates. The system provides notifications for interviews and job opportunities, giving students a personalized dashboard to manage their placement journey.

The Company Module allows recruiters to register, post and edit job openings, and manage applications efficiently. Companies can download resumes, Built-in analytics and notifications help optimize hiring strategies and streamline communication.

The Admin Module enables placement officers to manage student and company registrations, approve job posts, and coordinate placement drives. It supports interview scheduling, bulk notifications, and dispute resolution. Admins can also generate detailed placement reports for institutional analysis and planning.

Requirements for the Project

HTML5 CSS3 JavaScript PHP 8+ MySQL 5.7+ Bootstrap 5

The project uses HTML5, CSS3, and JavaScript for frontend development, styled with Bootstrap 5 for responsive UI. PHP 8+ handles server-side logic, authentication, and backend operations, while MySQL stores structured data such as student records, job postings, and interview schedules.

It uses PHPMailer for email notifications and PDO/MySQLi for secure database interactions. Session handling is managed using PHP Sessions or JWT for secure access control.

The system is best run on devices with a dual-core or higher processor and at least 4 GB RAM (8 GB recommended for development). It works across modern browsers (Chrome, Firefox, Safari, Edge), with a stable internet connection (1+ Mbps) required for real-time dashboard use.
