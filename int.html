Locatrix - Interactive Mapping

Project Details

The Interactive Map project was developed using HTML, CSS, JavaScript, and Leaflet.js to create a responsive and highly functional web mapping application. It allows users to add custom markers, plan routes, measure distances and areas, and explore categorized tourist locations across India.

The system integrates features like drawing tools, real-time location tracking, and reverse geocoding, enabling detailed spatial interactions. A modular UI with theme toggling, keyboard navigation, and clustering improves both usability and accessibility.

Requirements for the Project

HTML5 CSS3 JavaScript (ES6+) Leaflet.js

This project leverages modern frontend technologies including HTML5, CSS3, and JavaScript (ES6+) to create an interactive map-based web application. It uses the lightweight and mobile-friendly Leaflet.js library for map rendering and routing, along with plugins like Leaflet Draw for user annotations.

The system runs entirely on the client side and does not require a backend server. Browser-based LocalStorage is used to persist user data such as markers, map preferences, and annotations.
