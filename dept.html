Project Details

DEPTSYNC – A Smart Academic System is a web-based academic management platform designed to enhance communication and streamline workflows between students, staff, HODs, and the Principal. The system replaces traditional paper-based processes with digital solutions such as QR code-based attendance, leave approvals, assignment submissions, and real-time announcements.

Built using Flask (Python) and MySQL, DEPTSYNC provides role-specific dashboards and features for each user group. Students can view attendance, apply for leave, submit assignments, and access study materials from a unified dashboard. Staff can mark attendance via QR codes, review assignments, and manage academic materials.

The HOD module enables oversight of both student and staff activities, including leave approvals, attendance monitoring, and internal assessments. The Principal acts as the system admin, approving registrations and making announcements. The platform includes OTP-based registration, email notifications, and a secure approval workflow for new users.

Future enhancements include AI-powered academic analytics and chatbot assistance, aiming to identify academic trends, alert students at risk, and improve user interaction with intelligent suggestions and insights.

Requirements for the Project

HTML5 CSS3 JavaScript Python (Flask) MySQL Bootstrap 5

The frontend uses HTML5, CSS3, and JavaScript, styled with Bootstrap 5 for responsive user interfaces. QR code support is integrated for mobile devices with camera access. LocalStorage or SessionStorage is used to handle temporary user data for a smoother dashboard experience.

The backend is developed using Python (Flask), with MySQL for storing structured data like user profiles, attendance, assignments, and announcements. Flask extensions like Flask-Login and Flask-Session manage user authentication, while Flask-Mail and OTP systems handle secure registration and notifications. A QR code generation library is used for attendance functionality.

The system requires a modern device with at least a dual-core processor and 4 GB RAM (8 GB recommended). Users must have access to a webcam or phone camera for QR scanning and a stable internet connection (1+ Mbps) for real-time features. Developers should use a quad-core CPU, 8–16 GB RAM, and tools like VS Code/PyCharm, Postman, and MySQL Server for efficient development.
